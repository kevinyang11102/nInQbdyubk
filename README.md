# Python仓库管理系统 python717

## 项目概述

本项目是一款基于Python语言的仓库管理系统，采用前后端分离的架构设计，后端使用Django框架，前端使用Vue.js，数据库采用MySQL。系统旨在为用户提供一个高效、易用的仓库管理解决方案。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户管理
- 仓库管理
- 商品管理
- 库存管理
- 订单管理

## 系统角色

- 管理员：负责系统的管理与维护
- 员工：负责日常仓库操作的执行

## 运行环境

- 后端：Python 3.x，Django 3.x
- 前端：Node.js（版本需高于14）
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 准备运行环境
   - 安装Python 3.x
   - 安装Django 3.x
   - 安装Node.js（版本需高于14）
   - 安装MySQL 5.x 或更高版本

2. 克隆项目代码
   - 获取后端代码
   - 获取前端代码

3. 配置项目
   - 配置数据库
   - 配置前后端连接

4. 运行项目
   - 启动后端服务
   - 启动前端服务
